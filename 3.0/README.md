## Installation

* step 1- install node.js
* step 2- open command line and enter this file path
* step 3- input> npm install
* step 4- input> npm start
* step 5:- open browser and input url> localhost:3000
  
  
## Operating Manual

### snake system
    exit: esc  
    pause: p/ space  
    restart: r  

### red snake
    up: &ensp;↑  
    down: &ensp;↓  
    left: &ensp;←  
    right: &ensp;→  

### blue snake
    up: &ensp;w  
    down: &ensp;s  
    left: &ensp;a  
    right: &ensp;d
